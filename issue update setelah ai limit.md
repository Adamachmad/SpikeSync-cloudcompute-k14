Di dalam repositori, terdapat file inti Laravel di root direktori, namun ada juga sub-folder bernama volleytrack-saas yang membuat struktur proyek menjadi berantakan dan path Docker menjadi kebingungan.

Penyesuaian Path Docker: Periksa dan perbarui skrip docker-compose.yml dan Dockerfile yang ada. Pastikan semua volume mapping dan working directory mengarah ke root direktori (. atau /var/www/html), BUKAN ke dalam sub-folder volleytrack-saas.
